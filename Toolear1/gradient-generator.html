Linear Gradients: Angles and Directions

A **Linear Gradient** is defined by a straight line, called the gradient line, where the colors transition. The direction is controlled by either keywords (like to right) or a degree value (135deg).

Understanding Degrees

In CSS, gradients use a compass-like coordinate system for angles:

  • **0 degrees:** Transitions vertically from bottom to top.
  • **90 degrees:** Transitions horizontally from left to right.
  • **180 degrees:** Transitions vertically from top to bottom.
  • **135 degrees:** Transitions diagonally from bottom-left to top-right (a standard diagonal used in many modern UIs).
Our degree slider gives you precise control from 0 to 360 degrees, while the direction buttons offer quick access to the most common flows.

Color Stops and Smoothness

The color stops are the points where you define the color should be a certain value. In our tool, Color 1 is typically at the 0% stop, and Color 2 is at the 100% stop. By placing these stops, you create a seamless transition. For best visual quality, avoid large, contrasting color shifts over a very short distance, as this can lead to **color banding** (visible stripes instead of a smooth blend).

Use Case Examples

Website Hero Sections

A modern SaaS website needs a dynamic background that supports dark mode. Using a 135-degree gradient from a primary brand color to a slightly lighter complementary color creates depth and guides the user's eye towards the central text.

Call-to-Action (CTA) Buttons

Adding a subtle, vertical gradient (from 10% saturation to 15% saturation of the same color) to a button creates a tactile, three-dimensional appearance that signals clickability and improves user engagement.

Overlay for Readability

When placing white text over an image, the text can disappear over bright spots. A semi-transparent black gradient overlay (rgba(0,0,0,0.5) to transparent) ensures the text has a consistent dark background, making it easier to read.

Frequently Asked Questions

Are CSS gradients better than images?
For simple backgrounds, **yes**. CSS gradients are vectorsâ€”they are mathematically generated by the browser, meaning they load instantly, scale infinitely without pixelation, and take up only a few bytes of text file size. Using a large image for a simple blend is inefficient.
Why does 0 degrees go up?
CSS uses a standard mathematical convention, but it's rotated. **0 degrees** means the line starts at the bottom and points towards the top. **90 degrees** means the line starts on the left and points towards the right. This is reversed from standard compass headings.
Can I use transparency?
Yes. If you input a color with an alpha channel (like **RGBA**), the gradient will seamlessly transition to transparency. This is useful for fading elements into the background or creating soft shadows.
What is "Color Banding"?
Banding is a visual defect where a smooth gradient shows visible **stripes or bands** instead of a continuous blend. This usually happens over very large areas with limited color depth or very subtle color changes. It's a display issue, not a code issue, but using slightly stronger color stops can sometimes help.
